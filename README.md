# CustomerSupport-Chatbot

Developed an AI-powered FAQ Chatbot designed to automate customer support and provide instant responses to frequently asked questions. The chatbot was built using Python and implemented with AI and Machine Learning concepts to improve user interaction and response efficiency.

 Technologies & Tools Used:
• Python
• JSON for FAQ data management
• AI & NLP concepts
• VS Code
• Git & GitHub

What We Have Done:
• Built an intelligent chatbot for answering customer queries
• Created and managed FAQ datasets using JSON
• Implemented automated response handling
• Improved user interaction through AI-based responses
• Structured the project using modular Python programming
• Managed version control and collaboration using GitHub

This project helped in gaining practical experience in chatbot development, AI applications, problem-solving, and backend logic implementation.
